Anime Flow

Discover anime via a flowchart

Github

This project is derived from lukeatlook's anime recommendation flowchart*, which leads the user to a specific anime to watch through a series of questions.

Flowchart

This flowchart utilizes Jikan’s REST API, which allows for the searching and retriving of anime info from MyAnimeList. Jikan provides many other awesome anime features, so go check it out!

Jikan
1. Answer
Progress through the flowchart by answering each question
2. Results
Once you have answered all questions, you will be presented with one or more animes
3. Discover
After selecting an anime, Anime Flow will fetch the url** to view it on MyAnimeList!
* There is a slight change under the “fights” branch as most of those animes have finished. They are still in this flowchart, just in a different branch
** Anime Flow will display the top 3 results for that anime, possibly including OVA’s and movies. However, it may display other animes if there is only one result for the specified anime.
